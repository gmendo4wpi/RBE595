1. Install Airsim as in its website https://microsoft.github.io/AirSim/
2. Download Unreal 4.0
3. Download Stable-Baselines3 as in the documentation https://stable-baselines3.readthedocs.io/en/master/

-----------------------------------------------------
To run python files (Windows-10 only)

1. Open Blocks environment solution using Visual Studio 2022 
2. Press F5 to run the simulation using Unreal 4.0
3. Simply open a cmd terminal at this directory and run "Python main.py" to execute task3 as in the presentation.  
   The JSON file is already configured to nodisplay and overclocked for task3.
4. The file called task3 is just a short demo of the RQ library, the Main.Py file actually has all the DQN implementation.
